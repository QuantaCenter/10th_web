命令

命令 使用案例 选项/参数 描述
command --help / man ls --help / man [命令] - 帮助文档
ls ls [目录] 默认列出当前目录的非隐藏文件的文件名，a列出所有文件，l列出详细列表(ls -al别名 ll，此处注意选项可以叠加在一起)，添加目录参数则表示查看对应目录的内容 列出(list)目录内的文件
cd cd [目录] 默认返回用户主目录(~)，添加目录参数(可以使用相对或者绝对路径)则切换到该目录，如果没有那个目录则报错 切换目录(change directory)
pwd pwd - present work directory
cp cp 源文件 目标文件 r递归处理复制行为，a表示保留权限和链接并且递归处理复制 复制(copy)
mv mv 源文件 目标文件 - 改名或剪切(move)
touch touch [文件名] - 创建文件或者改变文件修改时间
rm rm [文件名] r循环删除(用于文件夹)，f强行删除(没有提示) 删除文件(remove)
mkdir mkdir [文件夹名] - 创建文件夹(make directory)
cat cat [文件名] - 在终端输出文件内容，tac表示逐行反向输出
more/less more [文件名] 或者 less [文件名] - 查看较长的文件，比较常用less
ps ps -aux常用 - 都是用于查询系统的进程
du du [文件/文件夹名] s查看所有文件的总大小，默认为当前文件夹下的所有文件的大小(会很长) 磁盘占用情况，默认单位为KB
df df [文件] 默认为整个文件系统的硬盘信息，加了文件参数表示查询该文件所在的磁盘分区信息，可以配合ln -s使用，判断是否在同一个分区 系统硬盘信息(不能查到Windows的)
lsof lsof lsof -i:port 表示查看该端口的占用情况，lsof -p 进程id，查看该进程的具体数据 用于查看你进程打开的文件，打开文件的进程，进程打开的端口(TCP、UDP)。找回/恢复删除的文件。
kill kill [进程号] - 用来删除执行中的程序或工作
date date [格式化参数，请自行搜索] - 日期
ln ln [文件] 默认为建立“硬链接”，会增加i-node的连接数(后面讲)，s为建立软链接，这个相当于Windows中的快捷方式 创建链接文件
free free - 内存信息
passwd passwd [用户] 默认为修改自己的密码，增加用户参数可以修改其他用户的密码(前提是你操作的是root用户) 密码修改
echo echo 字符串变量或者常量 - 通常可以打印出一些环境变量、命令执行的返回值之类的数据
clear clear - 清除终端的数据显示
grep grep [字符串或者正则表达式] [文件名] 默认为取得匹配表达式的结果的行并打印出来，n表示在取得行的同时加上对应行号，v表示把不匹配表达式的结果输出来，grep [字符串] -r [目录]表示在给定的目录下搜索所有文件中含有表达式的行 搜索文本
tar 很长，摸我 - tar包压缩和解压
zip zip [目标zip名，注意要加.zip后缀] [文件名] 默认只能压缩文件，q表示没有过程显示(静默执行)，r表示递归(用于文件夹的压缩) zip包压缩，如果同名的zip包已经存在，执行表示添加进去而不是覆盖
unzip zip [zip文件名] - zip包解压缩
scp scp 源文件路径 目的存放路径 默认只能复制文件，r可以递归复制(用于文件夹)。注意远程的表示方式：username@ip(host):/绝对路径 加密复制(secure copy)，类比cp，可以远程复制(前提是拥有用户权限和文件读权限)
gedit gedit [文件] - linux自带的很丑的编辑器
vi/vim vi 文件名 或 vim 文件名 - vi编辑器，功能强大
ssh ssh username@ip(host) 默认连接端口为22，如果端口不是22则需要用p选项+端口参数(eg, ssh xx@host -p port) 需要安装ssh服务才可以用，摸我
sudo sudo 命令 - 执行root用户或者其他用户才能执行的操作的时候，可以使用(前提是sudoer，如何设置？以后讲)
su su [用户名] 默认为切换到root用户，后面接了用户名表示切换到该用户(前提是要输入密码) 切换用户
apt-get apt-get 一些动作参数 [软件名] install安装，remove删除(保留配置文件)，update更新软件列表，upgrade更新已安装的软件 debian系的安装/更新/卸载软件的一个命令，是软件包管理系统
chmod chmod [权限设置的模式] [文件或文件夹] 默认为对该文件/文件夹进行权限改变，R对于文件夹表示对文件夹及以下的所有文件进行权限变更 权限变更(后面详细讲权限设置，这里不再赘述)。扩展阅读
chown chown 所有者:组名 文件 默认为对该文件/文件夹进行管理者改变，R对于文件夹表示对文件夹及以下的所有文件进行管理者变更 管理者变更(后面详细讲权限设置，这里不再赘述)。扩展阅读

操作

操作 使用案例 描述
> 命令 > 文件名(eg, ls -l > a.txt) 将输出内容重定向到文件中，不过如果文件本身存在，则会覆盖写入
>> 命令 >> 文件名(eg, ls -l >> a.txt) 将输出内容重定向到文件中，如果文件本身存在，则会追加写入
< 命令 < 文件名(eg, wc < a.txt，wc为统计文件/输入的行数、字数、字符数) 由文件来代替键盘输入
| 命令 | 命令(eg, ls | grep hello) 前一个命令的结果作为后一个命令的输入
& 命令 & 一般用于后台运行程序，扩展阅读
